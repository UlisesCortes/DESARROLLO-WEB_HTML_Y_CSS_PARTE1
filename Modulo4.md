## 6. Cómo se publica un sitio web en Internet (1)
Net2FTP es un cliente de FTP basado en el navegador web, es muy útil cuando se trabaja con equipos que no son propios, cuando el equipo es, por ejemplo, propiedad de la empresa en la que se trabaja. Net2FTP es rápido y no ocupa mucho espacio, con lo que no sobrecargará el navegador. Su ventaja es que funciona con cualquier sistema operativo. Se debe considerar evitar recursos que requieran descargas por tener que estar pendientes de actualizaciones o errores en la misma. Si bien winscp y smartftp son buenas opciones, al ser descargas perdemos funcionalidades o subsanación de errores si no se está pendiente de las actualizaciones.

Conceptos clave:
- Se Desarrollo en un Servidor de Desarrollo,
- Servidor de producción es uno com mejor atención a las peticiones de los Ususarios.
- FTP: Protocolo de transferencia de archivos, cliente de FTP.

## 7. Cómo se publica un sitio web en Internet (2): cliente FTP
El FTP o transferencia de archivos se usa para hacer transferencias de archivo de nuestro ordenador a un ordenador remoto, que será el servidor de FTP. Para hacer esta transferencia se utilizan programas llamados Cliente de FTP. Lo primero que hay hacer para publicar una web es acceder a un cliente de FTP e introducir los datos de la cuenta FTP que proporcionará el sitio de alojamiento donde tienes el hosting, así, establecerá la conexión. Una vez se conecta, se procede a subir los contenidos a la carpeta correspondiente, normalmente public_html, y se debe comprobar comprobar cómo se visualiza la web en Internet.

Conceptos clave:
Pasos para publicar unna pagina por FTP:
- Acceder a un cliente FTP
- Establecer la conexión con el FTP
- Subir los conteidos a la carpeta del servidor FTP
- Probar el Funcionamiento en Internet.

# Extras

Uno de los pasos más importantes a la hora de crea una página web es la selección del nombre. En la actualidad existen un gran número de páginas web con nombres o dominios registrados y, para duplicar nombres, tendremos que saber si el nombre que le queremos poner a nuestra página ya está siendo utilizado por otra empresa. Para obtener un nombre de dominio debemos dirigirnos al agente registrador, el cual nos indicará si el nombre que queremos utilizar está libre para que podamos usarlo en nuestra página web.

El alojamiento web o web hosting es el sistema que provee a los usuarios de Internet un espacio para poder almacenar información. Los productos que suelen ofrecer las empresas de alojamiento son los denominados planes o paquetes.

En la actualidad el acceso a Internet y a las páginas web no se reduce a las pantallas de ordenador, sino que se accede desde todo tipo de dispositivos móviles como smatphones o tablets. A la hora de calcular el tamaño de una página web, es esencial tener en cuenta el tamaño del código HTML, aparte de las hojas de estilo, el código JavaScript y de cualquier otro fichero que se utilice en la página.

A la hora de crear una página web es importante saber cuál es el objetivo que queremos con ella. Si queremos que la página web sea conocida, es necesario publicitarla en Internet. Las páginas web se desarrollan en un ordenador que se suele denominar servidor de desarrollo, pero este servidor no está permanentemente conectado a Internet, por lo que es más recomendable usar el servidor de producción, ya que este sí está preparado para atender las peticiones de los usuarios de forma concurrente.
